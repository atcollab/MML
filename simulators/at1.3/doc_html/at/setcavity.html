DESCRIPTION 

SETCAVITY - Set the RF cavity state
  ATCavityIndex = setcavity(InputString)

  INPUTS
  1. 'On', 'Off', or PassMethod {Default: no change}

  OUTPUTS
  1. ATCavityIndex - AT Index of the RF cavities
