SYNOPSIS

DESCRIPTION 

 GETCOD Closed Orbit Distortion
 GETCOD(RING,DP) finds closed orbit for a given momentum 
 deviation DP. It calls FINDORBIT4 which assumes a lattice
 with NO accelerating cavities and NO radiation
